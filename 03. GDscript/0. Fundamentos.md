# GDScript — Guía de Referencia Rápida (Godot 4.x) para Pythonistas

> 💡 **CHEATSHEET DE EQUIVALENCIAS PYTHON → GDSCRIPT**
> 
> |Concepto|Python|GDScript|
> |---|---|---|
> |Herencia de clase|`class Hijo(Padre):`|`extends Padre`|
> |Tipo de retorno|`def f() -> int:`|`func f() -> int:`|
> |Constante|`MAX = 100`|`const MAX = 100`|
> |Tipado estático|`x: int = 5`|`var x: int = 5`|
> |Nulo|`None`|`null`|
> |Imprimir|`print(x)`|`print(x)` ← igual|
> |Módulo (nativo)|`import math`|No hay imports — todo es global en Godot|
> |Decoradores|`@dataclass`|`@export`, `@onready`, `@tool`|
> |Match/Switch|`match` (3.10+)|`match` ← nativo|
> |Esperar async|`await coro()`|`await signal_o_callable`|
> |Sin cuerpo|`pass`|`pass` ← igual|
> |Self|`self`|`self` ← igual|
> |Super|`super().__init__()`|`super._ready()`|
