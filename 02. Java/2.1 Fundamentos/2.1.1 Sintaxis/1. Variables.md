[[1.1 Constantes e inmutabilidad]]
### Declaración de Variables, Literales y Ámbitos

#### 🔄 Contraste de Sintaxis

Esta es la diferencia más inmediata y constante entre Python y Java. En Python el tipo se **infiere siempre**. En Java debes **declararlo explícitamente** antes de usar cualquier variable. `var` (Java 10+) permite inferencia local, pero no es la norma en código de clase/universitario.

```
Python:              Java (clásico):         Java (var — Java 10+):
────────────         ─────────────────────   ──────────────────────
x = 10               int x = 10;             var x = 10;
nombre = "Ana"       String nombre = "Ana";  var nombre = "Ana";
precio = 9.99        double precio = 9.99;   var precio = 9.99;
activo = True        boolean activo = true;  var activo = true;
                     // ↑ minúscula en Java
```

**Ámbitos de variable (Scope) — más estricto que Python:**

java

```java
public class AmbitosDemo {

    // ÁMBITO DE CLASE (atributo): vive mientras el objeto exista
    // Equivale a self.x en Python dentro de __init__
    static int contadorGlobal = 0;   // atributo de clase (static)
    int id;                           // atributo de instancia

    public static void demostrarAmbitos() {

        // ÁMBITO DE MÉTODO: vive solo mientras se ejecuta el método
        int x = 10;   // existe desde aquí hasta la llave de cierre del método

        if (x > 5) {
            // ÁMBITO DE BLOQUE: vive solo dentro de estas llaves
            int dentroDelIf = 99;
            System.out.println(dentroDelIf); // ✅ accesible aquí
        }

        // ❌ System.out.println(dentroDelIf); // Error: variable fuera de ámbito

        for (int i = 0; i < 3; i++) {
            // 'i' solo existe dentro del for — esto es diferente a Python
            System.out.println(i);
        }
        // ❌ System.out.println(i); // Error: 'i' no existe aquí

        System.out.println(x); // ✅ x sigue accesible en el método
    }
}
```

#### 💻 Bloque de Código — Declaración y literales

java

```java
// LiteralesDemo.java
// Muestra todos los tipos de literales y sus sufijos obligatorios

public class LiteralesDemo {
    public static void main(String[] args) {

        // ── ENTEROS ──────────────────────────────────────────────────────
        int    decimal     = 42;
        int    conGuiones  = 1_000_000;   // Java 7+: _ como separador visual
        int    hexadecimal = 0xFF;        // prefijo 0x — igual que Python
        int    binario     = 0b1010;      // prefijo 0b — igual que Python
        long   grande      = 9_876_543_210L; // OBLIGATORIO el sufijo L para long

        // ── FLOTANTES ─────────────────────────────────────────────────────
        double precio    = 9.99;          // double por defecto (64 bits)
        float  pequeño   = 9.99f;         // OBLIGATORIO el sufijo f para float (32 bits)
        double cientifico = 1.5e10;       // notación científica — igual que Python

        // ── OTROS ─────────────────────────────────────────────────────────
        char   letra     = 'A';           // char: comillas SIMPLES, un solo carácter
        String texto     = "Hola";        // String: comillas DOBLES (siempre)
        boolean activo   = true;          // true/false en minúscula (Python: True/False)

        // En Python 'A' y "A" son equivalentes — en Java NO:
        // char usa ''  →  un solo carácter Unicode
        // String usa "" → secuencia de caracteres

        System.out.println("long:   " + grande);
        System.out.println("float:  " + pequeño);
        System.out.println("char:   " + letra);
        System.out.println("double: " + cientifico);
    }
}
```

#### 🪲 "Rompe el Código"

java

```java
// ❌ ERROR 1: olvidar el sufijo L en literales long
long poblacionMundial = 8000000000;   // ❌ error de compilación
// integer number too large — Java interpreta el literal como int,
// y 8.000.000.000 no cabe en int (max: ~2.100.000.000)
long poblacionMundial2 = 8_000_000_000L; // ✅ sufijo L obligatorio

// ❌ ERROR 2: comillas simples para String (hábito Python)
String nombre = 'Ana';   // ❌ error de compilación
// incompatible types: char cannot be converted to String
// En Java: '' = char (1 carácter), "" = String
String nombre2 = "Ana";  // ✅

// ❌ ERROR 3: variable no inicializada (Java no tiene valor por defecto local)
int resultado;
System.out.println(resultado);  // ❌ error de compilación
// variable resultado might not have been initialized
// En Python: NameError en runtime. En Java: error en COMPILACIÓN.
int resultado2 = 0;             // ✅ siempre inicializar variables locales
```
