### Tipos de Datos Primitivos (Los 8 Tipos)

#### 🔄 Contraste de Sintaxis

Python tiene un único tipo `int` (precisión arbitraria), un único `float` (64 bits), y `bool` como subclase de `int`. Java tiene **8 tipos primitivos** de tamaño fijo — no son objetos, viven en el Stack, y su tamaño en bits está garantizado en todas las plataformas.

|Tipo|Bits|Rango|Equivalente Python|
|---|---|---|---|
|`byte`|8|-128 a 127|`int` (acotado manualmente)|
|`short`|16|-32,768 a 32,767|`int` (acotado)|
|`int`|32|-2,147,483,648 a 2,147,483,647|`int` (el más usado)|
|`long`|64|±9.2 × 10¹⁸|`int` (precisión no arbitraria)|
|`float`|32|±3.4 × 10³⁸ (~7 decimales)|No existe directo|
|`double`|64|±1.8 × 10³⁰⁸ (~15 decimales)|`float`|
|`char`|16|0 a 65,535 (Unicode)|`str` de 1 carácter|
|`boolean`|1*|`true` / `false`|`bool`|

* El tamaño real de `boolean` en memoria depende de la JVM — se trata como 1 bit lógicamente.

**Wrappers (Clases envolventes)** — cuando necesitas un primitivo como objeto:

```
Primitivo → Wrapper (objeto en Heap):
int       → Integer
double    → Double
boolean   → Boolean
char      → Character
long      → Long
// etc.

// Autoboxing (Java 5+): conversión automática primitivo ↔ wrapper
int x = 42;
Integer xObj = x;        // autoboxing: int → Integer automáticamente
int xDe = xObj;          // unboxing: Integer → int automáticamente
```

#### 💻 Bloque de Código — Los 8 primitivos en práctica

java

```java
// OchoPrimitivos.java

public class OchoPrimitivos {
    public static void main(String[] args) {

        // ── byte: para datos pequeños, ahorra memoria en arrays grandes
        byte edadMinima = 18;
        byte edadMaxima = 127;
        System.out.println("byte máx: " + Byte.MAX_VALUE);   // 127

        // ── short: para datos que superan byte pero no necesitan int
        short añoActual = 2025;
        System.out.println("short máx: " + Short.MAX_VALUE);  // 32767

        // ── int: el entero por defecto — úsalo salvo razón específica
        int poblacion = 1_500_000;
        System.out.println("int máx: " + Integer.MAX_VALUE);  // 2,147,483,647

        // ── long: para números que superan int (IDs de BD, timestamps)
        long timestampUnix = 1_700_000_000_000L;  // milisegundos epoch
        System.out.println("long máx: " + Long.MAX_VALUE);

        // ── float: 32 bits, ~7 dígitos de precisión — evita en finanzas
        float precioFlotante = 19.99f;            // sufijo f obligatorio

        // ── double: 64 bits, ~15 dígitos — el flotante por defecto
        double precioDouble  = 19.99;             // sin sufijo = double
        System.out.printf("float:  %.10f%n", precioFlotante);  // ve el error de precisión
        System.out.printf("double: %.10f%n", precioDouble);

        // ── char: UN solo carácter Unicode (16 bits, sin signo)
        char inicial = 'J';
        char unicode = '\u00F1';   // ñ en Unicode
        char numero  = '7';        // ¡es char, no int!
        System.out.println("char: " + inicial + unicode); // Jñ

        // char + int = int (Java promueve tipos en operaciones)
        System.out.println(inicial + 1);    // 75 (ASCII de J es 74, +1 = 75)
        System.out.println("" + inicial + 1); // "J1" (String + concatenación)

        // ── boolean: solo true o false (no acepta 0/1 como Python)
        boolean esValido = true;
        boolean esMayor  = (poblacion > 1_000_000);  // expresión booleana
        System.out.println("válido: " + esValido);
        System.out.println("mayor:  " + esMayor);

        // ── Límites desde las clases Wrapper (equivale a sys.maxsize en Python)
        System.out.println("\n── Límites de tipos ──");
        System.out.println("int:    " + Integer.MIN_VALUE + " a " + Integer.MAX_VALUE);
        System.out.println("long:   " + Long.MIN_VALUE    + " a " + Long.MAX_VALUE);
        System.out.println("double: " + Double.MIN_VALUE  + " a " + Double.MAX_VALUE);
    }
}
```

#### 🪲 "Rompe el Código"

java

```java
// ❌ ERROR 1: desbordamiento (overflow) silencioso — Python nunca desborda
int maxInt = Integer.MAX_VALUE;   // 2,147,483,647
int overflow = maxInt + 1;         // ❌ NO lanza excepción — silenciosamente: -2,147,483,648
System.out.println(overflow);      // -2147483648  ← ¡resultado incorrecto sin aviso!

// En Python: 2_147_483_647 + 1 = 2_147_483_648  ← funciona perfectamente
// En Java: int tiene techo fijo — al superarlo, da vuelta al valor más negativo

// ✅ SOLUCIÓN: usar long cuando el valor puede superar Integer.MAX_VALUE
long sinOverflow = (long) maxInt + 1L;   // cast a long antes de operar
System.out.println(sinOverflow);          // 2147483648 ✅

// ❌ ERROR 2: usar boolean como si fuera int (hábito de C/Python mezclado)
boolean activo = true;
if (activo == 1) {    // ❌ error de compilación
    System.out.println("activo");
}
// incompatible types: int cannot be converted to boolean
// En Java, boolean NO es int. No existe equivalencia 0=false, 1=true a nivel de tipos.

// ✅ SOLUCIÓN:
if (activo) {         // ✅ boolean se usa directamente
    System.out.println("activo");
}
```
