#### 🔄 Contraste de Sintaxis

La mayoría de operadores aritméticos y de comparación son idénticos a Python. Las diferencias importantes:

|Operación|Python|Java|Diferencia|
|---|---|---|---|
|División entera|`//`|`/` entre ints|En Java `/` entre `int` ya es entera|
|Módulo|`%`|`%`|Igual|
|Potencia|`**`|`Math.pow(x,y)`|Java no tiene operador `**`|
|Y lógico|`and`|`&&`|Diferente palabra/símbolo|
|O lógico|`or`|`\|`|Diferente|
|Negación|`not`|`!`|Diferente|
|Diferente|`!=`|`!=`|Igual|
|Incremento|`x += 1`|`x++` o `++x`|Java tiene `++` y `--`|
|AND binario|`&`|`&`|Igual|
|OR binario|`\|`|`\|`|Igual|
|XOR binario|`^`|`^`|Igual (pero `^` no es potencia en Java)|

#### 💻 Bloque de Código — Operadores con sus trampas

java

```java
// OperadoresDemo.java

public class OperadoresDemo {
    public static void main(String[] args) {

        // ── ARITMÉTICOS ───────────────────────────────────────────────────
        int a = 17, b = 5;

        System.out.println(a + b);   // 22
        System.out.println(a - b);   // 12
        System.out.println(a * b);   // 85
        System.out.println(a / b);   // 3  ← división ENTERA (ambos son int)
        System.out.println(a % b);   // 2  ← módulo, igual que Python

        // Para división real, al menos uno debe ser double:
        System.out.println((double) a / b);  // 3.4 ← cast explícito
        System.out.println(17.0 / 5);        // 3.4 ← literal double

        // Potencia — Java NO tiene **
        double potencia = Math.pow(2, 8);    // 256.0 — siempre retorna double
        System.out.println((int) potencia);  // 256 ← cast si necesitas int

        // ── UNARIOS: ++ y -- (no existen en Python) ───────────────────────
        int x = 10;
        x++;          // post-incremento: x = x + 1 → x es ahora 11
        ++x;          // pre-incremento:  x = x + 1 → x es ahora 12
        x--;          // post-decremento: x = x - 1 → x es ahora 11
        System.out.println(x);  // 11

        // Diferencia pre vs post (importante en expresiones):
        int y = 5;
        int resultado1 = y++;   // resultado1 = 5 (valor ANTES del incremento), y = 6
        int resultado2 = ++y;   // resultado2 = 7 (valor DESPUÉS del incremento), y = 7
        System.out.println(resultado1 + " " + resultado2);  // 5 7

        // ── LÓGICOS: && y || tienen short-circuit igual que Python ─────────
        int contador = 0;
        boolean cond = (5 > 3) || (++contador > 0);  // || hace short-circuit
        System.out.println(contador);  // 0 — el segundo operando nunca se evaluó

        // ── OPERADORES DE ASIGNACIÓN COMPUESTA ────────────────────────────
        int n = 100;
        n += 50;    // n = 150  — igual que Python
        n -= 30;    // n = 120
        n *= 2;     // n = 240
        n /= 4;     // n = 60   ← división entera si n es int
        n %= 7;     // n = 4
        System.out.println(n);  // 4

        // ── OPERADORES BINARIOS (BIT A BIT) ───────────────────────────────
        int bits1 = 0b1010;   // 10 en decimal
        int bits2 = 0b1100;   // 12 en decimal

        System.out.println(bits1 & bits2);   // 8  (AND binario:  1000)
        System.out.println(bits1 | bits2);   // 14 (OR binario:   1110)
        System.out.println(bits1 ^ bits2);   // 6  (XOR binario:  0110)
        System.out.println(~bits1);          // -11 (NOT binario: complemento a 2)
        System.out.println(bits1 << 1);      // 20 (desplazamiento izquierda × 2)
        System.out.println(bits1 >> 1);      // 5  (desplazamiento derecha ÷ 2)
    }
}
```

#### 🪲 "Rompe el Código"

java

```java
// ❌ ERROR 1: esperar que / entre ints dé resultado decimal (hábito Python 3)
int total  = 7;
int partes = 2;
double mitad = total / partes;   // ❌ resultado: 3.0, no 3.5
// En Python 3: 7 / 2 = 3.5 siempre (/ siempre es división real)
// En Java: 7 / 2 = 3 (división entera entre int), LUEGO se convierte a 3.0

// ✅ SOLUCIÓN: castear ANTES de la división
double mitadCorrecta = (double) total / partes;   // 3.5 ✅

// ❌ ERROR 2: usar 'and', 'or', 'not' de Python
boolean condicion = (5 > 3) and (2 < 4);  // ❌ error de compilación
// ';' expected — 'and' no es una palabra reservada en Java

// ✅ SOLUCIÓN: operadores simbólicos
boolean condicionCorrecta = (5 > 3) && (2 < 4);  // ✅

// ❌ ERROR 3: confundir ^ (XOR binario) con ** (potencia Python)
int mal = 2 ^ 8;    // ❌ esto es XOR binario en Java, no 2^8=256
System.out.println(mal);  // 10 (2 XOR 8 = 0010 XOR 1000 = 1010 = 10)

// ✅ SOLUCIÓN para potencia:
double bien = Math.pow(2, 8);   // 256.0 ✅
```
