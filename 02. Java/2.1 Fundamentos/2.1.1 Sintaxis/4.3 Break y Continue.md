### Control de Saltos: `break` y `continue`

#### 🔄 Contraste de Sintaxis

`break` y `continue` funcionan igual que en Python. La adición exclusiva de Java son las **etiquetas (labels)** para romper o continuar loops externos desde un loop anidado — algo que Python no tiene y que evita el uso de variables bandera.

java

```java
// ── break y continue básicos (igual que Python) ───────────────────────────
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    System.out.print(i + " ");  // 0 1 2 3 4
}

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    System.out.print(i + " ");  // 1 3 5 7 9
}

// ── LABELS: break/continue a un loop externo específico ───────────────────
// Python no tiene esto — requeriría variable bandera o excepción

externo:                                    // etiqueta nombrada para el loop externo
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
        if (i + j == 7) {
            System.out.println("Par encontrado: (" + i + ", " + j + ")");
            break externo;                  // sale del loop EXTERNO completo
        }
    }
}
// Sin 'externo', el break solo saldría del for interno
// y el for externo continuaría — resultado incorrecto

// continue con etiqueta: salta a la siguiente iteración del loop externo
externo:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (j == 1) continue externo;   // salta al siguiente i, no al siguiente j
        System.out.println("i=" + i + " j=" + j);
    }
}
// i=0 j=0
// i=1 j=0
// i=2 j=0   ← j=1 y j=2 nunca se imprimen
```
