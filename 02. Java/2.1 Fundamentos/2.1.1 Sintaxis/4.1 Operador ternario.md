#### 🔄 Contraste de Sintaxis

La lógica es idéntica a Python, solo cambia el orden de la sintaxis:

```
Python: valor_si_true  if  condicion  else  valor_si_false
Java:   condicion  ?  valor_si_true  :  valor_si_false
```

java

```java
// Python:  estado = "mayor" if edad >= 18 else "menor"
// Java:
int edad = 20;
String estado = (edad >= 18) ? "mayor" : "menor";
System.out.println(estado);  // "mayor"

// Anidado (usar con moderación — sacrifica legibilidad igual que en Python)
int nota = 85;
String grado = (nota >= 90) ? "A"
             : (nota >= 80) ? "B"
             : (nota >= 70) ? "C"
             : "F";
System.out.println(grado);  // "B"

// Muy útil en inicializaciones compactas
int x = 10;
int absoluto = (x >= 0) ? x : -x;
```

#### 🪲 "Rompe el Código"

java

```java
// ❌ ERROR: usar ternario como sentencia standalone (sin consumir el valor)

int a = 5, b = 10;

(a > b) ? a : b;  // ❌ advertencia: expresión sin efecto — el valor se descarta

// ✅ SOLUCIÓN: siempre asignar o pasar como argumento el resultado
int mayor = (a > b) ? a : b;               // ✅ asignación
System.out.println((a > b) ? a : b);       // ✅ uso como argumento
```
