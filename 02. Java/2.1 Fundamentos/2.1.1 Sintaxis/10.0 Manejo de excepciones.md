# Resumen Rápido
```
| Try     | Intenta operar el bloque de codigo             |
| ------- | ---------------------------------------------- |
| Catch   | Agarra el error o la excepcion y lo muestra    |
| Finally | Se ejecuta siempre despues del try o del catch |
| Throw   | Lanza la exepción                              |
| Throws  | Declara las excepciones                        |
```
