## 🎭 Abstracción — Clases Abstractas e Interfaces

### 🔄 Contraste de Sintaxis

Python implementa abstracción con el módulo `abc`. Java tiene dos mecanismos distintos con propósitos complementarios:

|Característica|Clase Abstracta|Interfaz|
|---|---|---|
|Palabra clave|`abstract class`|`interface`|
|Instanciable|❌ No|❌ No|
|Herencia|Una sola (`extends`)|Múltiples (`implements`)|
|Puede tener estado|✅ Atributos de instancia|⚠️ Solo constantes (`static final`)|
|Métodos con cuerpo|✅ Sí|✅ `default` (Java 8+)|
|Métodos sin cuerpo|✅ `abstract`|✅ todos los no-default|
|Cuándo usar|Compartir **código** entre clases relacionadas|Definir un **contrato** de comportamiento|

**Analogía:** La clase abstracta es un "molde parcialmente rellenado". La interfaz es un "contrato firmado" que dice "quien me implemente, debe hacer esto".

```
Python ABC:                    Java Clase Abstracta:
────────────────               ──────────────────────────────────
from abc import ABC,           public abstract class Figura {
    abstractmethod
                                   // puede tener atributos y código real
class Figura(ABC):                 protected String color;
    @abstractmethod
    def area(self):                // método abstracto: sin cuerpo, subclase lo implementa
        pass                       public abstract double area();

                                   // método concreto: código real compartido
    def describir(self):           public void describir() {
        return "Soy figura"            System.out.println("Área: " + area());
                                   }
                               }
```

### 💻 Bloque de Código — Clase abstracta e interfaz en sistema real

```java
// ── INTERFAZ: define el CONTRATO de comportamiento ────────────────────────
// "Cualquier clase que implemente Serializable DEBE poder convertirse a JSON"

interface Serializable {
    String toJson();    // método abstracto implícitamente public abstract

    // Método default (Java 8+): implementación por defecto que las clases pueden sobreescribir
    default String toXml() {
        return "<objeto>" + toJson() + "</objeto>";
    }

    // Constante de interfaz: implícitamente public static final
    int VERSION_API = 1;
}

interface Imprimible {
    void imprimir();  // otro contrato — Java permite implementar múltiples interfaces
}

// ── CLASE ABSTRACTA: molde con código compartido ──────────────────────────
// No puede instanciarse directamente
abstract class Figura implements Serializable {

    // Atributos de instancia: las interfaces no pueden tener esto
    protected String color;
    protected String nombre;

    // Constructor de la clase abstracta (llamado por super() en subclases)
    public Figura(String nombre, String color) {
        this.nombre = nombre;
        this.color  = color;
    }

    // Método abstracto: cada subclase DEBE implementarlo con su propia fórmula
    public abstract double area();
    public abstract double perimetro();

    // Método concreto: código compartido por TODAS las subclases
    public void describir() {
        System.out.printf("Figura: %s | Color: %s | Área: %.2f | Perímetro: %.2f%n",
                          nombre, color, area(), perimetro());
    }

    // Implementa el contrato de Serializable
    @Override
    public String toJson() {
        return String.format("{\"nombre\":\"%s\",\"color\":\"%s\",\"area\":%.2f}",
                             nombre, color, area());
    }
}

// ── SUBCLASE CONCRETA: implementa los métodos abstractos ──────────────────
class Circulo extends Figura implements Imprimible {

    private double radio;

    public Circulo(double radio, String color) {
        super("Círculo", color);  // llama al constructor de Figura
        this.radio = radio;
    }

    // Obligatorio implementar los métodos abstractos de Figura
    @Override
    public double area() {
        return Math.PI * radio * radio;
    }

    @Override
    public double perimetro() {
        return 2 * Math.PI * radio;
    }

    // Obligatorio implementar el método de Imprimible
    @Override
    public void imprimir() {
        System.out.println("🔵 " + toJson());
    }
}

class Rectangulo extends Figura {

    private double base, altura;

    public Rectangulo(double base, double altura, String color) {
        super("Rectángulo", color);
        this.base   = base;
        this.altura = altura;
    }

    @Override
    public double area() { return base * altura; }

    @Override
    public double perimetro() { return 2 * (base + altura); }
}

// ── PROGRAMA PRINCIPAL ─────────────────────────────────────────────────────
class DemoAbstraccion {
    public static void main(String[] args) {

        // Figura f = new Figura("x", "rojo");  // ❌ Cannot instantiate abstract class

        Circulo    c = new Circulo(5.0, "rojo");
        Rectangulo r = new Rectangulo(4.0, 6.0, "azul");

        c.describir();    // usa el método concreto de Figura
        r.describir();

        System.out.println(c.toJson());   // implementación de Serializable
        System.out.println(c.toXml());    // método default de Serializable
        c.imprimir();                     // implementación de Imprimible

        // Polimorfismo via interfaz: cualquier Serializable puede toJson()
        Serializable[] objetos = {c, r};
        for (Serializable obj : objetos) {
            System.out.println(obj.toJson());
        }
    }
}
```

### 🪲 "Rompe el Código"

```java
// ❌ ERROR 1: no implementar todos los métodos abstractos

abstract class Animal {
    public abstract String sonido();
    public abstract String habitat();
}

class Perro extends Animal {
    @Override
    public String sonido() { return "Guau"; }
    // ❌ falta implementar habitat()
    // Error: Perro is not abstract and does not override abstract method habitat()
}

// ✅ SOLUCIÓN: implementar TODOS los métodos abstractos
class PerroCompleto extends Animal {
    @Override public String sonido()  { return "Guau";       }
    @Override public String habitat() { return "Doméstico";  }  // ✅
}


// ❌ ERROR 2: intentar tener atributo de instancia en interfaz

interface MiInterfaz {
    int contador = 0;       // parece un atributo — en realidad es public static final
    contador = 5;           // ❌ no es una sentencia válida dentro de una interfaz
    // Las interfaces NO tienen estado mutable de instancia
}

// ✅ Las interfaces solo tienen constantes (implícitamente public static final)
interface Constantes {
    int MAX_SIZE    = 100;   // ✅ public static final implícito
    String VERSION  = "1.0"; // ✅
}
```
