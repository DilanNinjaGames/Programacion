## 🧱 Conceptos Base — Clase, Objeto, Atributos, Métodos y Constructores

### 🔄 Contraste de Sintaxis

En Python una clase es un bloque `class` con `self` implícito y `__init__` opcional. En Java **todo código vive dentro de una clase** — no hay funciones sueltas, no hay scripts, no hay `self`: se usa `this`.

```
Python:                          Java:
──────────────────────────       ──────────────────────────────────────
class Perro:                     public class Perro {
    def __init__(self, nombre):
        self.nombre = nombre          private String nombre;   // atributo
        self.energia = 100            private int energia;

    def ladrar(self):                 // constructor
        print("Guau!")                public Perro(String nombre) {
                                          this.nombre  = nombre;
p = Perro("Rex")                          this.energia = 100;
p.ladrar()                            }

                                      public void ladrar() {
                                          System.out.println("Guau!");
                                      }
                                  }
                                  Perro p = new Perro("Rex");
                                  p.ladrar();
```

**Diferencias críticas:**

|Concepto|Python|Java|
|---|---|---|
|Referencia al objeto actual|`self` (nombre convención)|`this` (palabra reservada)|
|Constructor|`__init__`|mismo nombre que la clase|
|Instanciar|`Perro("Rex")`|`new Perro("Rex")`|
|Atributos de clase|`Perro.especie = "Canis"`|`static String especie = "Canis"`|
|Atributos de instancia|`self.nombre = nombre`|`this.nombre = nombre`|
|Sin constructor|Python usa `__init__` vacío|Java genera constructor por defecto|

### 💻 Bloque de Código — Clase completa con estado, comportamiento e identidad

```java
// CuentaBancaria.java
// Clase completa que demuestra: atributos, constructores, métodos y this

public class CuentaBancaria {

    // ── ATRIBUTOS DE CLASE (static): compartidos por TODAS las instancias ──
    private static int    totalCuentas  = 0;      // contador global
    private static double tasaInteres   = 0.05;   // 5% anual para todas

    // ── ATRIBUTOS DE INSTANCIA: únicos para cada objeto ───────────────────
    private String  titular;      // 'private' = solo accesible dentro de esta clase
    private String  numeroCuenta;
    private double  saldo;
    private boolean activa;

    // ── CONSTRUCTOR PRINCIPAL ─────────────────────────────────────────────
    // Mismo nombre que la clase, sin tipo de retorno (ni siquiera void)
    public CuentaBancaria(String titular, double saldoInicial) {

        // 'this.titular' = atributo de instancia
        // 'titular'      = parámetro del constructor
        // Sin 'this.' serían la misma variable — Java resuelve el conflicto con this
        this.titular      = titular;
        this.saldo        = saldoInicial;
        this.activa       = true;

        // Accedemos al atributo de CLASE con el nombre de la clase (más claro)
        CuentaBancaria.totalCuentas++;
        this.numeroCuenta = "CTA-" + totalCuentas;  // ej: CTA-1, CTA-2
    }

    // ── CONSTRUCTOR SOBRECARGADO: con saldo inicial por defecto ───────────
    // Java permite múltiples constructores con diferentes firmas
    public CuentaBancaria(String titular) {
        this(titular, 0.0);   // delega al constructor principal con saldo=0
        // 'this(...)' llama a OTRO constructor de la misma clase — debe ser
        // la PRIMERA línea del constructor
    }

    // ── MÉTODOS DE COMPORTAMIENTO ──────────────────────────────────────────
    public boolean depositar(double monto) {
        if (!this.activa || monto <= 0) return false;
        this.saldo += monto;
        return true;
    }

    public boolean retirar(double monto) {
        if (!this.activa || monto <= 0 || monto > this.saldo) return false;
        this.saldo -= monto;
        return true;
    }

    // Método que retorna información del estado (en Python: __str__)
    // En Java: toString() — se llama automáticamente en concatenaciones
    @Override
    public String toString() {
        return String.format("Cuenta[%s | Nro:%s | Saldo:$%.2f | %s]",
            titular, numeroCuenta, saldo, activa ? "Activa" : "Inactiva");
    }

    // ── MÉTODOS ESTÁTICOS: pertenecen a la clase, no a la instancia ───────
    public static int    getTotalCuentas() { return totalCuentas; }
    public static double getTasaInteres()  { return tasaInteres; }

    // Getters de instancia
    public String  getTitular()      { return this.titular; }
    public String  getNumeroCuenta() { return this.numeroCuenta; }
    public double  getSaldo()        { return this.saldo; }
    public boolean isActiva()        { return this.activa; }  // boolean: 'is' por convención

    // ── PROGRAMA DE PRUEBA ─────────────────────────────────────────────────
    public static void main(String[] args) {

        // Instanciar — 'new' crea el objeto en el Heap
        CuentaBancaria c1 = new CuentaBancaria("Ana García", 1000.0);
        CuentaBancaria c2 = new CuentaBancaria("Luis López");  // saldo=0

        System.out.println(c1);  // toString() se llama automáticamente
        System.out.println(c2);

        c1.depositar(500.0);
        c1.retirar(200.0);
        System.out.println("Saldo c1: $" + c1.getSaldo());  // 1300.0

        // Atributo de clase: accedido vía la clase, no la instancia
        System.out.println("Total cuentas: " + CuentaBancaria.getTotalCuentas()); // 2

        // Comparar identidad vs igualdad
        CuentaBancaria c3 = c1;              // c3 apunta al MISMO objeto
        System.out.println(c1 == c3);        // true  — misma referencia
        System.out.println(c1 == c2);        // false — objetos distintos
    }
}
```

### 🪲 "Rompe el Código"

```java
// ❌ ERROR 1: acceder a miembro de instancia desde método estático

public class BugEstatico {

    private String nombre = "Java";

    public static void main(String[] args) {
        System.out.println(nombre);  // ❌ non-static variable 'nombre'
                                     //    cannot be referenced from a static context
        // main() es static — no pertenece a ninguna instancia
        // 'nombre' es de instancia — necesita un objeto concreto para existir
    }

    // ✅ SOLUCIÓN 1: crear una instancia dentro de main
    public static void mainCorregido(String[] args) {
        BugEstatico obj = new BugEstatico();
        System.out.println(obj.nombre);  // ✅
    }

    // ✅ SOLUCIÓN 2: hacer el atributo static
    private static String nombreStatic = "Java";
    // ahora main() puede accederlo directamente
}


// ❌ ERROR 2: this() no es la primera línea del constructor

public class BugThis {
    int x, y;

    public BugThis(int x) {
        this.x = x;
        this(x, 0);  // ❌ call to this must be first statement in constructor
    }

    public BugThis(int x, int y) {
        this.x = x;
        this.y = y;
    }

    // ✅ SOLUCIÓN: this() SIEMPRE debe ser la primera sentencia
    public BugThisCorregido(int x) {
        this(x, 0);  // ✅ primera línea
        // código adicional puede ir después
    }
}
```
