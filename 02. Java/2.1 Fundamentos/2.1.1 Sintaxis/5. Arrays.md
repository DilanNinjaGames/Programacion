#### 🔄 Contraste de Sintaxis

Los arrays de Java son **fundamentalmente diferentes** a las listas de Python:

|Característica|Python `list`|Java `array`|
|---|---|---|
|Tamaño|Dinámico|**Fijo en creación — inmutable**|
|Tipos|Heterogéneo|**Homogéneo** (un solo tipo)|
|Índice negativo|✅ `lista[-1]`|❌ no existe|
|Métodos|`append`, `pop`, `sort`...|Solo `.length` (atributo)|
|Tamaño dinámico|`list` directamente|Usar `ArrayList<T>`|

**Tres formas de declarar:**

java

```java
int[] numeros  = new int[5];          // vacío, valores default 0
int[] pares    = {2, 4, 6, 8, 10};   // inicializado, tamaño inferido
int[] impares  = new int[]{1,3,5};   // explícito — equivale a la forma anterior
```

#### 💻 Bloque de Código — Arrays unidimensionales, multidimensionales y copia

java

```java
// ArraysDemo.java

import java.util.Arrays;

public class ArraysDemo {

    public static void main(String[] args) {

        // ── UNIDIMENSIONAL ────────────────────────────────────────────────
        double[] temperaturas = {22.5, 18.0, 25.3, 30.1, 19.8};

        // .length es un ATRIBUTO (sin paréntesis), no un método como len() en Python
        System.out.println("Cantidad: " + temperaturas.length);   // 5

        // Sin índices negativos — el último es length-1
        System.out.println("Última: " + temperaturas[temperaturas.length - 1]); // 19.8

        // Recorrido con for-each cuando solo necesitas el valor
        double suma = 0;
        for (double temp : temperaturas) {
            suma += temp;
        }
        System.out.printf("Promedio: %.2f°C%n", suma / temperaturas.length);

        // ── UTILIDADES DE Arrays ───────────────────────────────────────────
        int[] desordenado = {5, 2, 8, 1, 9, 3};

        Arrays.sort(desordenado);                          // ordena in-place O(n log n)
        System.out.println(Arrays.toString(desordenado));  // [1, 2, 3, 5, 8, 9]
        // ⚠️ println directo daría el hash del objeto, no el contenido

        int indice = Arrays.binarySearch(desordenado, 5);  // array DEBE estar ordenado
        System.out.println("Índice de 5: " + indice);      // 3

        int[] lleno = new int[5];
        Arrays.fill(lleno, 7);
        System.out.println(Arrays.toString(lleno));         // [7, 7, 7, 7, 7]

        // ── MULTIDIMENSIONAL ──────────────────────────────────────────────
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        System.out.println("Filas:    " + matriz.length);      // 3
        System.out.println("Columnas: " + matriz[0].length);   // 3
        System.out.println("Centro:   " + matriz[1][1]);       // 5

        // Recorrido de matriz
        for (int fila = 0; fila < matriz.length; fila++) {
            for (int col = 0; col < matriz[fila].length; col++) {
                System.out.printf("%3d", matriz[fila][col]);
            }
            System.out.println();
        }

        // Para imprimir array 2D completo:
        System.out.println(Arrays.deepToString(matriz));   // [[1,2,3],[4,5,6],[7,8,9]]

        // Matriz irregular (jagged): cada fila con distinto tamaño
        int[][] triangulo = new int[4][];
        for (int i = 0; i < triangulo.length; i++) {
            triangulo[i] = new int[i + 1];
            Arrays.fill(triangulo[i], i + 1);
        }
        for (int[] fila : triangulo) {
            System.out.println(Arrays.toString(fila));
        }
        // [1]  [2, 2]  [3, 3, 3]  [4, 4, 4, 4]

        // ── COPIADO DE ARRAYS ─────────────────────────────────────────────
        int[] original = {10, 20, 30, 40, 50};

        // Arrays.copyOf: copia los primeros n elementos
        int[] copia1 = Arrays.copyOf(original, 3);            // [10, 20, 30]

        // Arrays.copyOfRange: rango [desde, hasta)
        int[] copia2 = Arrays.copyOfRange(original, 1, 4);    // [20, 30, 40]

        // System.arraycopy: más eficiente, copia a nivel nativo
        // (origen, posOrigen, destino, posDestino, cantidad)
        int[] destino = new int[5];
        System.arraycopy(original, 0, destino, 0, original.length);
        System.out.println(Arrays.toString(destino));          // [10, 20, 30, 40, 50]
    }
}
```

// ❌ ERROR 2: intentar agregar elementos (los arrays tienen tamaño fijo)

numeros[3] = 4;     // ❌ ArrayIndexOutOfBoundsException — índice 3 no existe

// ✅ SOLUCIÓN 1: nuevo array más grande
int[] masBig = Arrays.copyOf(numeros, numeros.length + 1);
masBig[masBig.length - 1] = 4;

// ✅ SOLUCIÓN 2 (idiomática): usar ArrayList para colecciones dinámicas
import java.util.ArrayList;
ArrayList<Integer> lista = new ArrayList<>();
lista.add(1); lista.add(2); lista.add(3);
lista.add(4);  // ✅ ArrayList crece dinámicamente


// ❌ ERROR 3: comparar arrays con == o .equals()
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};
System.out.println(a == b);        // false — referencias distintas en el Heap
System.out.println(a.equals(b));   // false — equals no sobreescrito en arrays

// ✅ SOLUCIÓN:
System.out.println(Arrays.equals(a, b));           // true ✅ (1D)
System.out.println(Arrays.deepEquals(m1, m2));     // true ✅ (2D+)
