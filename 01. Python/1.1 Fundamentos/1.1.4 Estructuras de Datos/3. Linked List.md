Las listas enlazadas no son nada mas que nodos que están conectados unos con otros permitiendo que su alojamiento en la memoria se mucho mas efectivo, ya que no almacena todo en una misma linea de memoria, sino que lo guarda en diferentes segmentos.
En síntesis cada nodo tendrá un valor y el valor de la posición del nodo siguiente.

---
```Python
class Nodo:
	def __init__(self, valor):
		self.valor = valor
		self.siguiente = None

class ListaEnlazada:
	def __init__(self):
		self.cabeza = None
```
---

# Crear una lista enlazada
---
```Python
lista = ListaEnlazada()
nodo1 = Nodo(1) # <-- Creamos nuevo nodo
lista.cabeza = nodo1 # <-- Hacemos que sea la cabeza de la lista
# La cabeza es donde empiza la lista
nodo2 = Nodo(2)
nodo1.siguiente = nodo2 # <-- Le damos la direccion del nodo siguiente
nodo3 = Nodo(3)
nodo2.siguiente = nodo3
```
---

# Recorrer una lista enlazada
---
```Python
nodo = lista.cabeza # <-- Ingresamos a la lista
while(nodo.siguiente is not None):
	valor = nodo.valor # <-- Obtenemos el valor
	nodo = nodo.siguiente # <-- Accedemos al siguiente nodo
	print(valor)
```
