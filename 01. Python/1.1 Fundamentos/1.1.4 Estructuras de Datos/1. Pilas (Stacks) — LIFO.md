**Teoría:** Una **pila** es una estructura de datos que sigue el principio **LIFO: Last In, First Out** (Último en Entrar, Primero en Salir).

  

**Analogía perfecta:** Una pila de platos en la cocina. Solo puedes acceder al plato del **tope**. Para llegar al plato del fondo, tienes que ir retirando todos los de arriba. Al lavar un plato, lo colocas encima del tope.

  

Las dos operaciones fundamentales son:

- **`push`** (apilar): agrega un elemento al **tope** de la pila.

- **`pop`** (desapilar): elimina y retorna el elemento del **tope**.

  

**¿Por qué usar `list.append()` y `list.pop()` para implementar una pila?**

  

En Python, las listas son arrays dinámicos. Las operaciones `append()` y `pop()` (sin índice, opera en el último elemento) son **amortizadas $O(1)$** porque trabajan en el extremo final de la lista — no hay desplazamiento de elementos en memoria.

  

**Casos de uso reales:**

- El botón "Deshacer" (Ctrl+Z) de cualquier editor de texto.

- La pila de llamadas del intérprete de Python (el `traceback` que ves en errores).

- Evaluación de expresiones matemáticas con paréntesis.

- Navegación de historial de un navegador (botón "Atrás").

  

```python

# ══════════════════════════════════════════════════════════════════════════

# PILA (STACK) — Implementación con lista de Python

# ══════════════════════════════════════════════════════════════════════════

  

class Pila:

    """

    Implementación de una Pila (Stack) usando lista de Python.

  

    Todas las operaciones son O(1) amortizado.

    El 'tope' de la pila corresponde al ÚLTIMO elemento de la lista interna.

    """

  

    def __init__(self):

        self._datos = []   # la lista interna — el tope es el final

  

    def apilar(self, elemento) -> None:

        """push: agrega elemento al tope. O(1) amortizado."""

        self._datos.append(elemento)

  

    def desapilar(self):

        """pop: elimina y retorna el elemento del tope. O(1)."""

        if self.esta_vacia():

            raise IndexError("No se puede desapilar de una pila vacía")

        return self._datos.pop()   # pop() sin índice → último elemento

  

    def ver_tope(self):

        """peek: consulta el tope SIN eliminarlo. O(1)."""

        if self.esta_vacia():

            raise IndexError("La pila está vacía")

        return self._datos[-1]

  

    def esta_vacia(self) -> bool:

        """Retorna True si la pila no tiene elementos. O(1)."""

        return len(self._datos) == 0

  

    def tamano(self) -> int:

        """Número de elementos en la pila. O(1)."""

        return len(self._datos)

  

    def __repr__(self) -> str:

        if self.esta_vacia():

            return "Pila: [vacía]"

        tope = self._datos[-1]

        return f"Pila: {self._datos} ← tope={tope}"

  
  

# ── Prueba básica ─────────────────────────────────────────────────────────

pila = Pila()

print(pila)              # Pila: [vacía]

  

pila.apilar(10)

pila.apilar(20)

pila.apilar(30)

print(pila)              # Pila: [10, 20, 30] ← tope=30

  

print(pila.ver_tope())   # 30 (sin eliminar)

print(pila.desapilar())  # 30 — sale el ÚLTIMO que entró

print(pila.desapilar())  # 20

print(pila)              # Pila: [10] ← tope=10

print(pila.tamano())     # 1

  
  

# ── Caso real: Validador de paréntesis balanceados ────────────────────────

def validar_parentesis(expresion: str) -> bool:

    """

    Verifica que los paréntesis/corchetes/llaves estén correctamente balanceados.

    Usa una pila para rastrear los abiertos que esperan su cierre.

  

    Ejemplos válidos:   "(())", "{[()]}", "def f(): return (x + [1,2])"

    Ejemplos inválidos: "(]", "(()", "{[}"

    """

    pila = Pila()

  

    # Mapeamos cada cierre con su correspondiente apertura

    pares = {')': '(', ']': '[', '}': '{'}

    aperturas = set(pares.values())   # {'(', '[', '{'}

  

    for caracter in expresion:

  

        if caracter in aperturas:

            # Es una apertura → la apilamos esperando su par

            pila.apilar(caracter)

  

        elif caracter in pares:

            # Es un cierre → verificamos que el tope sea su par correcto

            if pila.esta_vacia():

                return False   # cierre sin apertura previa

  

            if pila.desapilar() != pares[caracter]:

                return False   # par incorrecto: ej. ( cerrado con ]

  

    # Al final, la pila debe estar vacía (toda apertura fue cerrada)

    return pila.esta_vacia()

  
  

casos = [

    ("(())",         True),

    ("{[()]}",       True),

    ("(()",          False),

    ("([)]",         False),

    ("{[}",          False),

    ("",             True),

    ("def f(): pass", True),

]

for expresion, esperado in casos:

    resultado = validar_parentesis(expresion)

    estado = "✅" if resultado == esperado else "❌"

    print(f"{estado} '{expresion}' → {resultado}")

  
  

# ── Caso real: deshacer acciones (Ctrl+Z) ────────────────────────────────

class EditorSimple:

    """Editor de texto minimalista con función deshacer."""

  

    def __init__(self):

        self.texto   = ""

        self._historial = Pila()   # guarda estados anteriores

  

    def escribir(self, texto_nuevo: str) -> None:

        self._historial.apilar(self.texto)   # guardamos estado actual

        self.texto += texto_nuevo

  

    def deshacer(self) -> bool:

        if self._historial.esta_vacia():

            print("Nada que deshacer")

            return False

        self.texto = self._historial.desapilar()

        return True

  

editor = EditorSimple()

editor.escribir("Hola")

editor.escribir(", mundo")

editor.escribir("!!!")

print(editor.texto)      # "Hola, mundo!!!"

editor.deshacer()

print(editor.texto)      # "Hola, mundo"

editor.deshacer()

print(editor.texto)      # "Hola"

```
