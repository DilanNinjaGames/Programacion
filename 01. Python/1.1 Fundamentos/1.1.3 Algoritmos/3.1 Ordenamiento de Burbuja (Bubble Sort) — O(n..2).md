**Teoría:** El nombre proviene de cómo los elementos más grandes "burbujean" o "flotan" hacia el final de la lista en cada pasada, igual que las burbujas de aire suben a la superficie del agua.

  

**¿Cómo funciona?** En cada **pasada** (iteración del bucle externo), el algoritmo recorre la lista comparando **pares de elementos adyacentes**. Si el elemento de la izquierda es mayor que el de la derecha, los intercambia. Al final de cada pasada, el elemento más grande del rango recorrido habrá "burbujeado" hasta su posición final correcta.

  

**Visualización de la primera pasada** sobre `[5, 3, 8, 1, 2]`:

  

```

[5, 3, 8, 1, 2]  → comparo 5 y 3: 5 > 3, INTERCAMBIO → [3, 5, 8, 1, 2]

[3, 5, 8, 1, 2]  → comparo 5 y 8: 5 < 8, sin cambio  → [3, 5, 8, 1, 2]

[3, 5, 8, 1, 2]  → comparo 8 y 1: 8 > 1, INTERCAMBIO → [3, 5, 1, 8, 2]

[3, 5, 1, 8, 2]  → comparo 8 y 2: 8 > 2, INTERCAMBIO → [3, 5, 1, 2, 8]

                                                                        ↑

                                                              8 llegó al final ✅

```

  

Después de la pasada 1: el 8 (el mayor) está en su posición definitiva. Necesitamos `n-1` pasadas para ordenar todo.

  

```python

# ══════════════════════════════════════════════════════════════════════════

# BUBBLE SORT

# ══════════════════════════════════════════════════════════════════════════

  

def bubble_sort(lista: list) -> list:

    """

    Ordena 'lista' usando el algoritmo de burbuja.

    MODIFICA la lista original in-place y también la retorna.

  

    Complejidad Temporal: O(n²) promedio y peor caso.

    Complejidad Espacial: O(1) — el intercambio es in-place.

    """

    n = len(lista)

  

    # Bucle externo: necesitamos n-1 pasadas para garantizar el orden

    # Después de la pasada i, los últimos i elementos ya están ordenados

    for pasada in range(n - 1):

  

        # Optimización: flag para detectar si ya está ordenado

        # Si en toda una pasada no hubo intercambio, la lista ya está ordenada

        hubo_intercambio = False

  

        # Bucle interno: compara pares adyacentes

        # Con cada pasada, el rango se reduce en 1 (el final ya está ordenado)

        for j in range(0, n - 1 - pasada):

  

            # ¿Está el par en el orden incorrecto?

            if lista[j] > lista[j + 1]:

  

                # Intercambio (swap) usando unpacking de Python — elegante y O(1)

                lista[j], lista[j + 1] = lista[j + 1], lista[j]

                hubo_intercambio = True

  

        # Optimización: si no hubo intercambios, la lista ya está ordenada

        # → Cortamos el algoritmo temprano (mejor caso: O(n) para lista ya ordenada)

        if not hubo_intercambio:

            break

  

    return lista

  
  

# ── Pruebas ────────────────────────────────────────────────────────────────

numeros = [64, 34, 25, 12, 22, 11, 90]

print(f"Original:  {numeros}")

print(f"Ordenada:  {bubble_sort(numeros)}")

  
  

# ── Versión con traza completa ────────────────────────────────────────────

def bubble_sort_con_traza(lista: list) -> list:

    lista = lista.copy()   # no modificamos el original para la demo

    n = len(lista)

    total_intercambios = 0

  

    for pasada in range(n - 1):

        hubo_intercambio = False

        print(f"\n── Pasada {pasada + 1} ──")

  

        for j in range(0, n - 1 - pasada):

            if lista[j] > lista[j + 1]:

                lista[j], lista[j + 1] = lista[j + 1], lista[j]

                hubo_intercambio = True

                total_intercambios += 1

                print(f"  Intercambio [{j}]↔[{j+1}]: {lista}")

  

        if not hubo_intercambio:

            print(f"  ✅ Sin intercambios — lista ordenada en {pasada+1} pasadas")

            break

  

    print(f"\nResultado: {lista} ({total_intercambios} intercambios totales)")

    return lista

  

bubble_sort_con_traza([5, 3, 8, 1, 2])
```
