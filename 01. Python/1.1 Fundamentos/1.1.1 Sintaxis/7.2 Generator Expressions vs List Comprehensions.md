# ── Sintaxis: solo cambian los corchetes por paréntesis ───────────────────

lista_comp = [x**2 for x in range(10)]   # list — todo en memoria

gen_expr   = (x**2 for x in range(10))   # generator — lazy

  

# ── Cuándo usar cada uno ──────────────────────────────────────────────────

# ✅ Lista: cuando necesitas acceder múltiples veces, indexar, o conocer el largo

# ✅ Generador: cuando solo iteras UNA vez y el conjunto es grande

  

# Los generadores se pueden pasar directamente a funciones que aceptan iterables

total    = sum(x**2 for x in range(1_000_000))   # sin [] extra — más eficiente

promedio = sum(x for x in range(100)) / 100

  

# ── El generador se "agota" — solo se puede iterar una vez ────────────────

gen = (x for x in range(3))

print(list(gen))   # [0, 1, 2]

print(list(gen))   # []   ← ya se agotó, no hay valores nuevos

  

# Para re-iterar, hay que re-crear el generador
