### 2.1 Conversión de Bases y Manipulación Numérica


```python

# ── Conversión de bases numéricas ─────────────────────────────────────────

print(bin(42))          # '0b101010'  → binario

print(hex(255))         # '0xff'      → hexadecimal

print(oct(8))           # '0o10'      → octal

  

# Inverso: int(cadena, base) convierte DESDE cualquier base

print(int('101010', 2))  # 42  (binario → decimal)

print(int('ff', 16))     # 255 (hex → decimal)

print(int('10', 8))      # 8   (octal → decimal)

  

# ── Funciones numéricas esenciales ────────────────────────────────────────

print(abs(-7.5))          # 7.5   — valor absoluto

print(round(3.14159, 2))  # 3.14  — redondeo con precisión

print(round(2.5))         # 2     ← Python usa "banker's rounding" (al par)

print(round(3.5))         # 4

  

# pow(base, exp) es igual a **, pero pow(base, exp, mod) es más eficiente

# para criptografía (no computa el número completo antes del módulo)

print(pow(2, 10))         # 1024

print(pow(2, 10, 1000))   # 24   ← equivale a (2**10) % 1000 pero más rápido

  

# divmod: cociente Y resto en una sola llamada (evita hacer // y % por separado)

cociente, resto = divmod(17, 5)

print(cociente, resto)    # 3  2

  

# Uso real de divmod: convertir segundos a horas/minutos/segundos

segundos_totales = 3725

horas, rem       = divmod(segundos_totales, 3600)

minutos, segs    = divmod(rem, 60)

print(f"{horas}h {minutos}m {segs}s")   # 1h 2m 5s

```

### 2.2 Funciones Integradas de Iteración


```python

numeros = [4, 1, 9, 2, 7, 3]

  

# ── len, sum, min, max ────────────────────────────────────────────────────

print(len(numeros))        # 6

print(sum(numeros))        # 26

print(min(numeros))        # 1

print(max(numeros))        # 9

  

# Con key= (aplican una función antes de comparar)

palabras = ["banana", "kiwi", "manzana"]

print(min(palabras, key=len))   # "kiwi"  ← la más corta

print(max(palabras, key=len))   # "manzana"

  

# ── sorted() → nueva lista ordenada (no modifica el original) ─────────────

print(sorted(numeros))               # [1, 2, 3, 4, 7, 9]

print(sorted(numeros, reverse=True)) # [9, 7, 4, 3, 2, 1]

  

# Ordenar lista de tuplas por el segundo elemento

personas = [("Ana", 30), ("Luis", 25), ("Mia", 35)]

print(sorted(personas, key=lambda p: p[1]))

# [('Luis', 25), ('Ana', 30), ('Mia', 35)]

  

# ── reversed() → iterador invertido (no crea lista nueva) ─────────────────

for item in reversed(numeros):

    print(item, end=" ")   # 3 7 2 9 1 4

# Para obtener lista: list(reversed(numeros))

  

# ── enumerate() → pares (índice, valor) ──────────────────────────────────

frutas = ["manzana", "pera", "uva"]

for i, fruta in enumerate(frutas, start=1):   # start=1 para base 1

    print(f"{i}. {fruta}")

# 1. manzana

# 2. pera

# 3. uva

  

# ── zip() → combina iterables en paralelo (se detiene en el más corto) ────

nombres = ["Ana", "Luis", "Mia"]

edades  = [30, 25, 35]

for nombre, edad in zip(nombres, edades):

    print(f"{nombre}: {edad}")

  

# Truco: zip(*lista_de_listas) transpone una matriz

matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

transpuesta = list(zip(*matriz))

print(transpuesta)   # [(1,4,7), (2,5,8), (3,6,9)]

```

### 2.3 Expresiones Lambda


Una lambda es una **función anónima de una sola expresión**. No admite múltiples líneas ni sentencias complejas. Su valor real está en **pasarlas como argumento** a otras funciones.


```python

# ── Sintaxis: lambda argumentos: expresión ────────────────────────────────

cuadrado = lambda x: x ** 2

suma      = lambda x, y: x + y

  

print(cuadrado(5))    # 25

print(suma(3, 4))     # 7


# NOTA: PEP 8 desaconseja asignar lambdas a variables (usa def en su lugar).

# El uso correcto es pasarlas INLINE como argumento.


# ── map(): aplica función a cada elemento ─────────────────────────────────

numeros = [1, 2, 3, 4, 5]

cuadrados = list(map(lambda x: x**2, numeros))

print(cuadrados)   # [1, 4, 9, 16, 25]

# Equivalente con list comprehension (más pythonico):

cuadrados = [x**2 for x in numeros]


# ── filter(): filtra elementos según condición ────────────────────────────

pares = list(filter(lambda x: x % 2 == 0, numeros))

print(pares)   # [2, 4]

# Equivalente:

pares = [x for x in numeros if x % 2 == 0]

  

# ── sorted() con key= (el uso más importante de lambda) ──────────────────

empleados = [

    {"nombre": "Ana",  "salario": 50000},

    {"nombre": "Luis", "salario": 35000},

    {"nombre": "Mia",  "salario": 70000},

]

  

# Ordenar por salario ascendente

por_salario = sorted(empleados, key=lambda e: e["salario"])

# Ordenar por nombre descendente

por_nombre  = sorted(empleados, key=lambda e: e["nombre"], reverse=True)

  

# Clave de ordenamiento múltiple (tupla): primero por dept, luego por salario

datos = [("ventas", 50000), ("it", 70000), ("ventas", 40000), ("it", 60000)]

ordenado = sorted(datos, key=lambda x: (x[0], x[1]))

print(ordenado)

# [('it', 60000), ('it', 70000), ('ventas', 40000), ('ventas', 50000)]

```
