> **Orden de ejecución real de un SELECT (≠ orden de escritura):**

> ```

> FROM → JOIN → WHERE → GROUP BY → HAVING → SELECT → DISTINCT → ORDER BY → LIMIT

> ```

> ```sql

> -- Esqueleto completo de una query

> SELECT   columnas / funciones

> FROM     tabla

> JOIN     otra_tabla ON condicion

> WHERE    filtro_de_filas

> GROUP BY columna_agrupadora

> HAVING   filtro_de_grupos

> ORDER BY columna ASC|DESC

> LIMIT    n;

> ```
