### 2.1 `INSERT INTO` — Insertar Filas

  

#### ⚙️ Mecánica de la Query

  

```

INSERT INTO ──► Valida que la tabla exista

               ──► Valida tipos de datos por columna

                   ──► Valida restricciones (NOT NULL, CHECK, FK)

                       ──► Escribe la fila en el segmento de datos

                           ──► Actualiza todos los índices afectados

                               ──► Registra operación en el log de transacciones ✅

```

  

El motor inserta atómicamente: si la validación de cualquier restricción falla, **ninguna fila** se inserta (todo o nada).

  

#### 🔤 Sintaxis Estándar

  

```sql

-- Inserción explícita (recomendada — especifica columnas)

INSERT INTO tabla (col1, col2, col3)

VALUES (val1, val2, val3);

  

-- Múltiples filas en una sola sentencia (más eficiente)

INSERT INTO tabla (col1, col2)

VALUES

    (val1a, val2a),

    (val1b, val2b),

    (val1c, val2c);

  

-- Insertar desde resultado de SELECT (INSERT ... SELECT)

INSERT INTO tabla_destino (col1, col2)

SELECT col_a, col_b

FROM   tabla_origen

WHERE  condicion;

```

  

#### 🗄️ Categoría Estricta

**DML** — Manipula filas (datos), no la estructura.

```sql

-- Insertar categorías (tabla padre primero — por la FK)

INSERT INTO categorias (nombre, descripcion, activa) VALUES

    ('Electrónica',    'Dispositivos y gadgets tecnológicos',   TRUE),

    ('Ropa',           'Prendas de vestir para todas las edades', TRUE),

    ('Hogar',          'Artículos para el hogar y decoración',  TRUE),

    ('Deportes',       'Equipamiento deportivo y fitness',      TRUE),

    ('Descontinuados', 'Productos fuera de catálogo',           FALSE); -- inactiva

  

-- Verificar las categorías insertadas

SELECT id_categoria, nombre, activa FROM categorias;

  

-- Insertar productos vinculados a sus categorías

INSERT INTO productos (nombre, precio, stock, id_categoria) VALUES

    ('Laptop Pro 15"',      1299.99, 50,   1),  -- Electrónica

    ('Auriculares Bluetooth', 89.99, 200,  1),  -- Electrónica

    ('Camiseta Algodón',      19.99, 500,  2),  -- Ropa

    ('Zapatillas Running',    79.99, 150,  4),  -- Deportes

    ('Lámpara LED',           34.99, 300,  3),  -- Hogar

    ('Teclado Mecánico',      149.99, 75,  1),  -- Electrónica

    ('Pantalón Deportivo',    39.99, 250,  4);  -- Deportes

  

-- INSERT ... SELECT: copiar productos sin stock a una tabla de archivo

CREATE TABLE IF NOT EXISTS productos_sin_stock (

    LIKE productos INCLUDING ALL  -- PostgreSQL: copia estructura completa

);

  

-- MySQL equivalente:

CREATE TABLE productos_sin_stock AS

    SELECT * FROM productos WHERE stock = 0;  -- copia estructura Y datos filtrados

```
