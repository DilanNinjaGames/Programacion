### 5.1 Funciones de Agregación

#### ⚙️ Mecánica de la Query

Las funciones de agregación **colapsan múltiples filas en un único valor resumen**. Operan sobre conjuntos de filas, no sobre filas individuales.

  

```

Sin GROUP BY → agrega TODAS las filas en UN solo resultado:

  SELECT COUNT(*) FROM productos;   → un número (ej: 7)

  

Con GROUP BY → agrega POR GRUPO, un resultado POR grupo:

  SELECT id_categoria, COUNT(*) FROM productos GROUP BY id_categoria;

  → una fila por cada categoría distinta

  

Manejo de NULL en funciones de agregación:

  COUNT(*)        → cuenta todas las filas (incluyendo NULLs)

  COUNT(columna)  → cuenta filas donde columna NO es NULL

  SUM(columna)    → ignora NULLs en la suma

  AVG(columna)    → calcula el promedio SIN contar los NULLs en el divisor

```

  

#### 🔤 Sintaxis Estándar

  

```sql

SELECT

    COUNT(*)          AS total_filas,        -- cuenta todo, incluye NULLs

    COUNT(columna)    AS total_no_nulos,     -- excluye NULLs

    COUNT(DISTINCT c) AS total_unicos,       -- valores únicos

    SUM(columna)      AS suma,

    AVG(columna)      AS promedio,

    MIN(columna)      AS minimo,

    MAX(columna)      AS maximo,

    GROUP_CONCAT(col  SEPARATOR ', ') AS lista_concatenada  -- MySQL

FROM tabla;

```

  

#### 🗄️ Categoría Estricta

**DQL** — Las funciones de agregación son parte del `SELECT`.

  

#### 💻 Laboratorio Práctico

  

```sql

-- ══════════════════════════════════════════════════════

-- LAB 11: Funciones de Agregación

-- ══════════════════════════════════════════════════════

  

-- Estadísticas globales de toda la tabla productos

SELECT

    COUNT(*)                AS total_productos,      -- todas las filas

    COUNT(id_categoria)     AS con_categoria,         -- las que NO son NULL

    COUNT(DISTINCT id_categoria) AS categorias_usadas, -- cuántas categorías distintas

    SUM(stock)              AS stock_total,

    ROUND(AVG(precio), 2)   AS precio_promedio,      -- redondear a 2 decimales

    MIN(precio)             AS precio_minimo,

    MAX(precio)             AS precio_maximo,

    MAX(precio) - MIN(precio) AS rango_precio        -- expresión entre funciones

FROM productos;

  

-- COUNT(*) vs COUNT(columna): la diferencia con NULLs

-- Insertar un producto sin categoría para demostrar

INSERT INTO productos (nombre, precio, stock, id_categoria)

VALUES ('Producto Sin Cat', 25.00, 10, NULL);

  

SELECT

    COUNT(*)              AS cuenta_todas_las_filas,   -- incluye el NULL

    COUNT(id_categoria)   AS cuenta_solo_con_categoria -- excluye el NULL

FROM productos;

-- total_productos puede ser 8, con_categoria puede ser 7

  

-- GROUP_CONCAT: lista de productos por categoría en una sola celda

SELECT

    c.nombre        AS categoria,

    GROUP_CONCAT(p.nombre ORDER BY p.nombre SEPARATOR ' | ') AS productos_lista

FROM categorias c

INNER JOIN productos p ON c.id_categoria = p.id_categoria

GROUP BY c.id_categoria, c.nombre;

-- Resultado: Electrónica | Auriculares Bluetooth | Laptop Pro 15" | Teclado Mecánico

```
