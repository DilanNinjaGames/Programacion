### 1.2 `ALTER` — Modificar Estructuras Existentes

  

#### ⚙️ Mecánica de la Query

  

`ALTER` modifica los **metadatos** de un objeto ya existente. Dependiendo de la operación, el motor puede necesitar **reescribir físicamente** la tabla completa (ej. cambiar un tipo de dato) o solo actualizar el catálogo (ej. renombrar una columna). En tablas grandes, algunos `ALTER` pueden tardar minutos.

  

```

ALTER TABLE ──► Localiza tabla en catálogo

               ──► Bloquea la tabla (lock)

                   ──► Ejecuta la modificación estructural

                       ──► Actualiza metadatos en catálogo

                           ──► Libera el lock ✅

```

  

#### 🔤 Sintaxis Estándar

  

```sql

-- Agregar columna

ALTER TABLE tabla ADD COLUMN nueva_col TIPO_DATO RESTRICCIONES;

  

-- Eliminar columna

ALTER TABLE tabla DROP COLUMN nombre_col;

  

-- Renombrar columna

ALTER TABLE tabla RENAME COLUMN viejo_nombre TO nuevo_nombre;

  

-- Modificar tipo/propiedades de columna

ALTER TABLE tabla MODIFY COLUMN columna NUEVO_TIPO;    -- MySQL

ALTER TABLE tabla ALTER COLUMN columna TYPE NUEVO_TIPO; -- PostgreSQL

  

-- Agregar restricción

ALTER TABLE tabla ADD CONSTRAINT nombre_restriccion

    FOREIGN KEY (columna) REFERENCES otra_tabla(col);

  

-- Eliminar restricción

ALTER TABLE tabla DROP CONSTRAINT nombre_restriccion;

```

  

#### 🗄️ Categoría Estricta

**DDL** — Modifica estructura, no filas.

  

#### 💻 Laboratorio Práctico

  

```sql

-- ══════════════════════════════════════════════════════

-- LAB 2: ALTER — Evolucionar la tabla productos

-- ══════════════════════════════════════════════════════

  

-- Agregar columna de descuento (en %)

ALTER TABLE productos

    ADD COLUMN descuento DECIMAL(5, 2) DEFAULT 0.00;

  

-- Agregar columna de descripción larga

ALTER TABLE productos

    ADD COLUMN descripcion TEXT;

  

-- La columna 'stock' ahora debe ser NOT NULL y tener mínimo 0

ALTER TABLE productos

    MODIFY COLUMN stock INT NOT NULL DEFAULT 0;

  

-- Renombrar 'fecha_alta' a 'fecha_creacion' para mayor claridad

ALTER TABLE productos

    RENAME COLUMN fecha_alta TO fecha_creacion;

  

-- Agregar una restricción CHECK para que el descuento esté entre 0 y 100

ALTER TABLE productos

    ADD CONSTRAINT chk_descuento CHECK (descuento BETWEEN 0 AND 100);

  

-- Eliminar el índice de precio anterior y crear uno más específico

DROP INDEX idx_precio ON productos;

CREATE INDEX idx_precio_categoria ON productos(precio, id_categoria);

  

-- Verificar la nueva estructura

DESCRIBE productos;

```
