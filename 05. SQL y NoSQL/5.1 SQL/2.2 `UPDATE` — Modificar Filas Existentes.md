### 2.2 `UPDATE` — Modificar Filas Existentes

  

#### ⚙️ Mecánica de la Query

  

```

UPDATE ──► Localiza las filas que cumplen el WHERE

          (si no hay WHERE: afecta TODAS las filas ⚠️)

          ──► Para cada fila encontrada:

              ──► Valida las restricciones del SET

                  ──► Escribe el nuevo valor en disco

                      ──► Actualiza índices afectados

                          ──► Registra en log de transacciones ✅

```

  

**Regla de oro:** Siempre escribe y valida el `WHERE` **antes** de ejecutar el `UPDATE`. Muchos DBAs ejecutan primero un `SELECT` con el mismo `WHERE` para confirmar qué filas se afectarán.

  

#### 🔤 Sintaxis Estándar

  

```sql

-- Actualización simple

UPDATE tabla

SET    columna1 = nuevo_valor1,

       columna2 = nuevo_valor2

WHERE  condicion;

  

-- Actualización con expresión (basada en el valor actual)

UPDATE tabla

SET    precio = precio * 1.10   -- aumentar precio un 10%

WHERE  id_categoria = 1;

  

-- Actualización con subconsulta

UPDATE tabla

SET    columna = (SELECT valor FROM otra_tabla WHERE condicion)

WHERE  condicion_principal;

```

  

#### 🗄️ Categoría Estricta

**DML** — Modifica datos en filas existentes.

  

#### 💻 Laboratorio Práctico

  

```sql

-- ══════════════════════════════════════════════════════

-- LAB 5: UPDATE — Actualizar precios y stock

-- ══════════════════════════════════════════════════════

  

-- PASO 1: Siempre verificar ANTES con SELECT qué filas se afectarán

SELECT id_producto, nombre, precio

FROM   productos

WHERE  id_categoria = 1;          -- Electrónica: 3 productos

-- Resultado: Laptop Pro, Auriculares, Teclado Mecánico

  

-- PASO 2: Aplicar aumento del 10% a todos los productos de Electrónica

UPDATE productos

SET    precio = precio * 1.10     -- multiplica el precio actual por 1.10

WHERE  id_categoria = 1;          -- ← CRÍTICO: filtrar solo Electrónica

  

-- PASO 3: Verificar el resultado

SELECT id_producto, nombre, precio

FROM   productos

WHERE  id_categoria = 1;

-- Laptop Pro: 1299.99 → 1429.989 (aprox 1430.00)

  

-- Aplicar descuento del 15% a productos con stock > 200

UPDATE productos

SET    descuento = 15.00          -- 15% de descuento

WHERE  stock > 200;

  

-- Reducir stock en 1 unidad al producto con id=1 (simulación de venta)

UPDATE productos

SET    stock = stock - 1

WHERE  id_producto = 1

  AND  stock > 0;                 -- doble seguridad: no permitir stock negativo

  

-- Actualización de múltiples columnas a la vez

UPDATE productos

SET    precio     = 99.99,

       descuento  = 20.00,

       stock      = 100

WHERE  id_producto = 3;           -- Camiseta Algodón

  

-- Verificar estado final

SELECT nombre, precio, descuento, stock

FROM   productos

ORDER BY id_producto;

```
